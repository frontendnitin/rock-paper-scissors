# ✂️ Rock Paper Scissors

A fun Rock Paper Scissors game built with vanilla HTML, CSS and JavaScript.

## 🔴 Live Demo
[rock-paper-scissors-drab-ten.vercel.app](https://rock-paper-scissors-drab-ten.vercel.app)

## ✨ Features

- 🎮 Play against the computer
- 🏆 Score tracking
- 🖼️ Visual icons for rock, paper and scissors
- 🔄 Reset game option

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML | Structure |
| CSS | Styling |
| JavaScript | Game logic |

## 🚀 Run Locally

**1. Clone the repository:**
```bash
git clone https://github.com/frontendnitin/rock-paper-scissors.git
```

**2. Open `index.html` in your browser** ✅
